    Streams

    Jun 25, 2021

    What is a Stream?

    A Workflow Stream is a way of directing the output of AireForms/forms4health to a Workflow. The AireFlow plugin is already installed in forms4health, so collecting the output of a form can be accomplished simply by using the correct routing key. A stream will then trigger the associated workflow with the form as input data.

    Routing Key

    The routing key can be arbitrary text that identifies one or more Streams. To link with forms4health, this is set by convention as {FORM_KEY}-{VERSION}. These values are defined within forms4health. For example, if you had version 3 of a form with key about-me, then the routing key you should use is about-me-3.
