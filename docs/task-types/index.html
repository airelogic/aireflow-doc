    Task Types

    Feb 10, 2022

    Manual

    A manual task is used for when someone needs to perform an action e.g. make an observation.

    Email

    An email task is as it sounds. An email is sent to a user (perhaps defined in the input data). While intended to inform the user that there is a task they need to complete, it can also be used to send receipt of a submission, a summary reply, or any other email. For example, a manager might receive an email notifying them of a ticket that has been raised, while the customer might receive a notification that their issue has been assigned to a manager to look into.

    HTTP

    An HTTP task is a simple POST request to a defined url. The body of this request can be formed to include data and match any required shape for the receipient endpoint.
