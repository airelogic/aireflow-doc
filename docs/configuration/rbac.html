RBAC ​

What is RBAC? ​

Role-based access control (RBAC) restricts access to certain functionality in AireFlow based on your role within your organisation.

Roles ​

There are currently 4 suggested roles. You may have more than one of these roles assigned to your user account:

Workflow Builder ​

  • View, create, import, and delete workflows.
  • View, create, edit, delete, and sync repositories.
  • View, create, edit, and delete event definitions.

Runtime ​

  • View, edit, and delete task instances.
  • Ability to fire events.

Runtime Debugger ​

  • View event logs.
  • View workflow logs.
  • Delete the event context from the event logs.

Tenant Admin ​

  • View, create, edit, and delete tenants.
